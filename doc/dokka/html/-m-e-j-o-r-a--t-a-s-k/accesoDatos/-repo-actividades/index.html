RepoActividades

class RepoActividades(val actividades: MutableList<Actividad> = mutableListOf(), val tareas: MutableList<Tarea> = mutableListOf(), val eventos: MutableList<Evento> = mutableListOf()) : IActividadRepository

Esta clase sirve para poder gestionar las actividades, dando opciones para añadir actividades y guardarlas en un fichero o cargar las actividades de dicho fichero, el cual se encuentra en la ruta descrita por la variable RUTA_FICHERO_ACTIVIDADES.

También permite cambiar el estado de una tarea.
